int VC0706_ChildInit ( void  )

Child Task Initialization function.

This function is invoked during VC0706 application startup initialization to create and initialize the Camera Child Task. The purpose for the child task is to send commands to the camera, and to recieve then store images taken.

Returns:
The success value of the child task creation function call

VC0706_AppInit
void VC0706_ChildTask ( void  )

The entry point for the VC0706 application child task.

Registers with CFE as a child task, interfaces with the parent task and calls the child task main loop function. Should the main loop function return due to a breakdown in the interface handshake with the parent task, this function will self delete as a child task with CFE.
